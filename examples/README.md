# shodan-ts Examples

This directory contains a few examples demonstrating how to use the `shodan-ts` library.

### Examples Index

* `01-basic-search.ts` - Searching a keyword and looping through IPs.
* `02-list-ports-and-cves.ts` - Listing all open ports and CVEs for a specific IP found by Shodan.
* `03-dns-recon.ts` - Resolving a domain.
* `04-subdomain-enumeration.ts` - Bulk fetching subdomains, resolving their IPs, and querying open ports.

### Contribute an Example
New examples are always welcome to help grow this collection! Whether written in TypeScript or plain JavaScript, if you have an interesting use case or a unique way of leveraging the Shodan API, please consider contributing an example.

Check out the [Contribution Guidelines](../CONTRIBUTING.md) to learn how to submit a Pull Request and add a script to this list!
